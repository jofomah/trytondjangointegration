Tryton Django Integration

This is just a proof of concept that shows how Tryton ERP server can be integrated with a Django Web client, While Proteus acts as the connector between the two.

This project has just party and module web interface,

  • Party: This is used to represent anything you want to see as an entity in your system, it can be used to model a supplier, customer, organisation, etc. Party has the following properties, Name, code(a tag), language, address(which can be more than one, though we made provision for just one in this sample app)

  • Module: This is just a simple module manager interface that can be used to install Trytond modules that have been installed on the Tryton server module folder via pip install trytond-module-name.
